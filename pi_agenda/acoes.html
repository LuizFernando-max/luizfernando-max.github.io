Ações JavaScript

Duas funcionalidades específicas do sistema de Agenda Comercial, descritas, estilizadas e implementadas com JavaScript.

01

Filtro de Profissionais por Serviço

Ao selecionar um serviço, apenas os profissionais habilitados para aquela especialidade são exibidos — impedindo agendamentos inválidos.

⚙️
Funcionalidade

Cada profissional possui um array especialidades[] com os IDs dos serviços que ele atende. Ao trocar o serviço selecionado, a função filtrarProfissionais() percorre todos os profissionais e renderiza apenas os que contêm aquele serviço na lista.

🎯
Aplicabilidade

Em clínicas, salões e consultórios com equipes multidisciplinares, esse filtro garante que o cliente nunca veja um médico como opção para um serviço de estética — tornando o sistema confiável e profissional.

💡
Exemplo de uso

Selecione "Barbearia" abaixo: apenas Bruno Melo aparece. Troque para "Psicologia": apenas Elena Costa. "Odontologia": apenas Ana Lima, e assim por diante.

02

Verificação de Conflito de Horário

Ao combinar profissional + data, o sistema filtra em tempo real quais horários estão livres ou já reservados — impedindo o duplo agendamento.

⚙️
Funcionalidade

A função verificarConflitos(profId, dataStr) filtra o array AGENDA[] buscando entradas com o mesmo profissional e data. Os horários encontrados são marcados como indisponíveis. Slots livres aceitam clique; slots ocupados exibem tooltip e são bloqueados visualmente.

🎯
Aplicabilidade

Garante integridade da agenda mesmo antes da integração com o banco de dados MySQL. Essa lógica será replicada no backend Node.js futuramente, mas já funciona no frontend para simulações e demos.

💡
Exemplo de uso

Selecione "Bruno Melo" e a data de hoje: os horários já "agendados" aparecem bloqueados. Troque para "Ana Lima": outros horários estarão ocupados. Clique num slot livre para simular um novo agendamento e veja o slot ser bloqueado.
